## Installation

### Using python3.9 environment
```sh
python3.9 -m venv .kvasir_seg_venv
source .kvasir_seg_venv/bin/activate
git clone https://github.com/GerasimovIV/kvasir-seg.git
cd kvasir-seg
pip install -r requirements.txt
```
and download datasets
```sh
make datasets
```
